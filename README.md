# Retro-Light-Cycle-Game

## Overview
This project showcases the retro arcade "Tron" game, a passion of mine are the Tron movies created by Disney. Hence this is a passion project to test my JavaScript skills by creating a project related to something I enjoy.

This project will showcase Javascript, CSS, and HTML skills to create the game within a web browser

## Result
This code will produce a copy of the retro "Tron" light cycle game, with a user controlled cycle and an AI controlled one with each user win resulting in points and eventually providing a high score.

## Tools Used
- HTML
- JavaScript
- CSS
